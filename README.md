Inlämningsuppgift 1 för C#/.NET fortsättningskurs.

Inlämningen går ut på att skapa APIer för ett forum där anävndare ska kunna registreras och loggas in.
Det ska gå att skriva inlägg
Det ska gå att kommentera inlägg.
Inläggen ska skunna skapas, läsas, uppdateras och tas bort.
Kommentarerna ska kunna skapas, läsas, uppdateras och tas bort.
Användare ska kunna skapas, läsas, uppdateras och tas bort.
